This dataset contains the experimental setup and results of the evaluation of TypeWriter on the open-source data set:

 * github_urls.txt: List of all projects in the dataset.

 * open_source_training_files.txt: Files used for training the model.

 * open_source_validation_files.txt: Files used for validating the model.

 * open_source_results_argument.json and open_source_results_return.json: Type predictions by the TypeWriter model. For each type slot, we list the developer-provided annotation and the top-5 predictions by the model.
